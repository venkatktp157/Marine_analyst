This project is a **Marine Data Analyst AI Agent** built with [LangChain](https://www.langchain.com/), [Groq](https://groq.com/), and [Streamlit](https://streamlit.io/).  
It allows you to upload **marine data files** (PDF, Word, Excel, or images like PNG/JPEG/BMP) and interactively query them through a web interface.  
The agent summarizes and analyzes the content in the role of a **Marine Data Analyst**, providing insights relevant to marine science and operations.
